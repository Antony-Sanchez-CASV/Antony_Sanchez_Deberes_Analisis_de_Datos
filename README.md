# Antony_Sanchez_Deberes_Analisis_de_Datos
Trabajo en clase 
El objetivo es contar el número de ocurrencias de las palabras en un archivo de texto. El resultado final deberá ser similar a:

(harry,250) 

(potter,153) 

(niño,48 )
